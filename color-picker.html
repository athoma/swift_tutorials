Color Picker

In this project you'll learn about using the Picker View and Structs

Check out the Structs and Picker View documentation if you get stuck.

1. Open up a new Single View application

In the StoryBoard, add a Picker View, and a Label. You may place them wherever you’d like, but make sure that they are Horizontally Aligned, and have a set Vertical Spacing between them.

2. Use the Assistant Editor to link your Picker View and Label elements to your ViewController as Global elements.

3. Create a new Swift File, and inside it create a Struct called “Colors” that has contains a String Constant “colorName”, and a UIColor Constant “backgroundColor”. Save the file, and return to your ViewController.

4. Create a global variable called “colorsArray”, typecast it to be an array of “Colors”, and set it equal to an empty array. Create a function called “loadColors” that returns nothing and takes in no parameters.

5. Inside the loadColors function, add at least three colors to our colorsArray. Then initialize loadColors inside viewDidLoad.

6. Now we have an array with all of our colors, but we still need to link them to our Picker View. For this, we will need to use some of the Picker View functions which can be found here and here.

In this project we will need:

func numberOfComponents(in pickerView: UIPickerView) -> Int to choose how many columns our Picker View has

func pickerView(_ pickerView: UIPickerView, numberOfRowsInComponent component: Int) -> Int { to choose how many rows our Picker View has

func pickerView(_ pickerView: UIPickerView, titleForRow row: Int, forComponent component: Int) -> String? so that we can rename each row of the Picker View

func pickerView(_ pickerView: UIPickerView, didSelectRow row: Int, inComponent component: Int) so that we can control what happens if a specific row is selected

7. For now, we only want one column in our Picker View, so have numberOfComponents return 1.

Inside numberOfRowsInComponent, return the number of elements that we have in our colorsArray without using an actual number.

8. Inside titleForRow, we are going to use the “row: Int” property to return each of our colorNames. So just as you would select a specific element of your array with an integer, use “row” in place of that integer to return all of the colorNames in our colorsArray.

9. In didSelectRow, use that same “row: Int” property to change the color of our background when selected (you may need to look up how to change the background color). Also change the text of our label to equal the colorName of whatever color is selected.

10. In viewDidLoad, set the delegate and dataSource of the Picker View to “self”. Then preload the color of the background to equal the first color in our array, as well as the text in the label.
